# **App Name**: Monumenta Projects Homepage

## Core Features:

- Hero Section: Full-width hero with a background image of a restored monument, overlaid with text and CTA buttons to guide users to key actions.
- Who We Are: Concise introductory text, providing a brief overview of Monumenta Projects and linking to the About page for more detailed information.
- Key Pillars Grid: A grid of cards representing the core services, highlighting Monumenta's expertise in heritage conservation, ASI projects, infrastructure, and documentation.
- Featured Projects Carousel: Showcase a selection of completed conservation projects with images, titles, locations, and descriptions, linking to individual project pages.
- Trust and Differentiation Points: Present a bulleted list of key differentiators, emphasizing experience with ASI standards, use of traditional materials, and commitment to local communities.
- CSR Partnership Block: Inform corporations on how they can fulfill CSR goals through Monumenta's work; include a CTA to learn more about partnership opportunities.
- Contact Strip: A concluding call to action inviting potential clients to send their project details via a prominently placed button.

## Style Guidelines:

- Primary color: Light brown (#A67B5B) to evoke a sense of earthiness and history.
- Background color: Sandstone beige (#F5F0E1) to reflect Rajasthani architecture. 
- Accent color: Antique gold (#D4A373) for highlights and call-to-action elements.
- Headings font: 'Playfair' serif for titles, headings (heritage, premium feel).
- Body font: 'Lora' serif for body text, descriptions, longer copy (heritage, premium feel).
- UI/Buttons: 'Poppins' sans-serif font (modern, CSR-friendly).
- Line icons for key pillars and trust points.
- Responsive grid layout for content sections. Use a mobile-first approach for all devices.
- Subtle fade-in and scroll animations to enhance user experience.
